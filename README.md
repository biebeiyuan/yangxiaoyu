# 🎮 无期迷途主题互动解谜游戏

一个基于《无期迷途》世界观的互动式网页解谜游戏，玩家扮演局长，通过探索虚拟桌面系统，揭开隐藏在现实背后的真相。

## 🌟 游戏特色

- 🖥️ **沉浸式桌面模拟** - 完整还原 Windows 桌面环境
- 📱 **多应用交互** - 微信、钉钉、OA系统、文件管理器等
- 🔍 **线索收集** - 在各个应用中寻找隐藏的线索
- 📖 **多重结局** - 根据玩家选择触发不同结局
- 🎵 **隐藏彩蛋** - 包含背景音乐和特殊互动

## 🎯 游戏玩法

1. 从桌面开始，点击各个应用图标探索
2. 阅读邮件、聊天记录、文件等寻找线索
3. 解开密码锁，查看隐藏文件夹
4. 阅读完所有日记触发最终选择
5. 根据你的决定进入不同结局

## 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (原生)
- **存储**: localStorage (用于解锁状态)
- **部署**: Vercel
- **特性**: 
  - 响应式设计
  - 长按交互
  - 音频播放
  - 动态内容加载

## 🚀 本地运行

### 方法一：使用 Python HTTP 服务器（推荐）

```bash
# 进入项目目录
cd 000_test

# 启动服务器
python -m http.server 8000

# 或直接运行批处理文件
start_server.bat
```

然后访问 `http://localhost:8000`

### 方法二：直接打开

双击 `index.html` 文件（可能有部分功能受限）

## 🌐 在线部署

查看 [部署指南](DEPLOY.md) 了解如何部署到 Vercel。

快速部署：
```bash
npm install -g vercel
vercel --prod
```

## 🎮 游戏提示

### 密码线索
- **劳动合同**: 包含入职时间信息
- **便签纸**: 密码格式提示
- **日记**: 核心剧情线索

### 隐藏内容
- 🔒 加密文件夹需要正确密码
- 📧 QQ邮箱中有重要线索
- 💬 微信搜索功能可发现隐藏内容
- 🎵 结局页面右下角有音频彩蛋

### 触发结局
阅读完「个人备份」文件夹中的所有日记后，会自动触发最终选择。
